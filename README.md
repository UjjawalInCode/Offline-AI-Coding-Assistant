# Offline AI Coding Assistant

## Overview

An offline AI-powered coding assistant developed using ASP.NET Core MVC and LM Studio.

The system integrates a locally hosted Large Language Model (Qwen2.5-Coder) to provide:

- Code generation
- Code debugging
- Code explanation
- Technical assistance

## Technologies Used

- ASP.NET Core MVC
- C#
- LM Studio
- Qwen2.5-Coder
- REST API
- Git & GitHub

## Architecture

User → ASP.NET MVC → AIService → LM Studio API → Qwen Model

## Features

- Offline AI inference
- Local network deployment
- Multi-device access
- Secure intranet operation
- Code generation and debugging
